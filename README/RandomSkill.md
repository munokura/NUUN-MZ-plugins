発動するスキルがランダムに発動出来るスキルを設定できます。  

## 設定
スキルのメモ欄  
`<RandomSkill:[id]>` 指定IDで設定したスキルからランダムに発動するスキルを設定します。  
`[id]`:プラグインパラメータのランダム発動スキル設定のリスト番号。
