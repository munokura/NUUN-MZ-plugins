アイテム、スキルの文字色の色を変更できます。  
![画像](img/ItemNameColor_1.png)  
![画像](img/ItemNameColor_2.png)

### 設定方法
アイテム、スキルのメモ欄  
`<NameColor:[id]>` 文字の色を指定します。[id]にはカラーインデックスを記入します。    
`<NameColor:16>` 文字色をカラーインデックス16番に変更します。  
`<NameColor:[#RRGGBB]>` カラーコードで文字色を変更します。  
`<NameColor:#00bfff>`
