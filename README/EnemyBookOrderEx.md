#### 名前順の設定
以下のタグがない場合はモンスター名で比較されます。  
漢字がある場合、`EnemyOrder`で読み方を設定してください。  

敵キャラのメモ欄
`<EnemyOrder:[name]>`  
`[name]`:読み方
