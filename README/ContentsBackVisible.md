#### 必須、前提プラグイン
[共通処理](https://github.com/nuun888/MZ/blob/master/README/Base.md)  

コンテンツの背景を表示しないようにします。  

コンテンツ背景クラス設定はコンテンツ背景非表示がOFFの時は適用するクラスのみ設定します。  
ONの時は適用しないクラスを指定します。  

他のプラグインでコンテンツ背景に任意の画像を指定する場合、該当のクラスのコンテンツ背景非表示を適用しないようにしてください。
