[共通処理](https://github.com/nuun888/MZ/blob/master/README/Base.md)   

メニューコマンドの表示順を任意の表示順に表示できます。  
表示順はプラグインパラメータのメニューコマンド設定での設定したID順に表示されます。  
未設定のコマンドは設定したコマンドの後に表示されます。
