通常の会心攻撃の倍率に乗算されます。  

## 設定
会心力は特殊能力値扱いになります。  
特徴を有するメモ欄  
`<CreEffectRate:[rate]>`   
[rate]:会心効果率  

取得パラメータ
Game_BattlerBase  
`crp`
