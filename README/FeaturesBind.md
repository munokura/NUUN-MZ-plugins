交代できない及び逃走出来なくなる特徴を設定できます。  

## 設定
特徴を有するメモ欄  
`<FeaturesBind>`   
この特徴のあるアクターはメンバー変更がすることが出来ません。  
この特徴のあるアクターが戦闘メンバーに含まれている場合、逃走することが出来ません。  
この特徴がある敵は逃走することが出来ません。
