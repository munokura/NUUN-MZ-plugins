#### 必須、前提プラグイン
[共通処理](https://github.com/nuun888/MZ/blob/master/README/Base.md)  

マップの上限数(デフォルト2000)、アニメーションの上限数(デフォルト1000)を超えて設定できるようになります。  
上限の変更を行うにはプラグインコマンドでテストプレイ中に実行します。  
実行前にバックアップをとることを推奨いたします。
