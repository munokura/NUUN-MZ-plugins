## オプションパラメータ名
リストの番号に設定した文字列が変数、スイッチに対応するIDとなります。  
変数1:リスト0番  
スイッチON:リスト0番(OFFの場合は1番)  

## キー、ゲームパッドのキー(ボタン)変更の操作
キー、ゲームパッドのキー、ボタンの変更は変更したい項目でEnter(決定ボタン)を押し、任意のキー(ボタン)を押します。  
既に設定済みのキー(ボタン)、予約キー(ボタン)は設定できません。
