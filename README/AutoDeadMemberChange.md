メンバーが戦闘不能になった時に自動でメンバーを交代します。  

## 設定
### 戦闘不能毎に交代
デフォルトでは全滅時に交代になっておりますので戦闘不能毎に交代する場合は、戦闘不能毎に交代に指定のスイッチIDを指定してtureに設定してください。
