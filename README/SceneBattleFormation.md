戦闘中にメンバーを変更できるようにします。  
このプラグインはメンバー変更画面（NUUN_SceneFormation）の拡張機能です。  

![画像](img/SceneFormation2.png)  

※アクターコマンドの並び替え条件は選択不可時のときは現在、暫定的に不透明に表示する仕様にしてあります。
