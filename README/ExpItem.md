経験値を増減させるアイテムやスキルを作ることが出来ます。

## 経験値増減アイテム、スキルの設定
アイテム、スキルのメモ欄  
`<ExpIncrease:[exp]>` 経験値を増減させます。  
`<ExpIncrease:10000>` 経験値が10000増加します。  
`<ExpIncrease:-5000>` 経験値が5000減少します。  
`<levelUpStop>` 次のレベルに上がるまで経験値が増加します。  
`<NolevelDown>` 現在のレベルの経験値が0になるまで経験値が減少します。  

## メッセージフォーマット
経験値増加時減少時のメッセージ  
%1:使用者 %2：対象者 %3：経験値  
効果なしの時のメッセージ  
%1:使用者 %2：対象者
